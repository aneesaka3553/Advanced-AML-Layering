# 🚀 Advanced AML Compliance Dashboard

## 📌 Overview
This project demonstrates an end-to-end KYC (Know Your Customer) and AML (Anti-Money Laundering) compliance dashboard built using Excel and Power BI.

The dashboard simulates real-world banking operations by analyzing client data, risk factors, and transaction patterns to identify high-risk customers and improve compliance monitoring.

## 🧠 Key Features

### ✅ Layered AML Risk Model
Implemented a multi-layer risk scoring system:
- Customer Risk (PEP, Income)
- Geographic Risk (Country-based)
- Transaction Risk (Amount, Flagged)
- Behavioral Risk (Patterns & frequency)

### ✅ Risk Classification
- Low Risk (0–50)
- Medium Risk (51–100)
- High Risk (101+)

### ✅ KYC Monitoring
- Tracks KYC completion status
- Identifies onboarding bottlenecks

### ✅ Transaction Monitoring
- Detects suspicious transactions
- Flags high-risk activities

## 📈 Dashboard Components

- KPI Cards:
  - Total Clients
  - KYC Completion %
  - High Risk Clients
  - Flagged Transactions
  - Average Risk Score

- Visualizations:
  - Client Risk Segmentation
  - Risk Layer Contribution
  - High Risk by Geography
  - Transaction Trend Analysis
  - Suspicious Activity Detection

- Interactive Filters:
  - Country
  - Risk Category
  - KYC Status
